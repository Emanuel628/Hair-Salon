# Project Brief

## Project Name

Hair Salon Website Concept

## Created By

**Mejor Tech**

**Slogan:** Building Better Technology That Makes Sense

## Purpose

This project is a premium interactive website concept for hairstylists, salons, and beauty-service professionals.

The goal is to move beyond the standard salon website pattern of:

- Basic hero image
- Service list
- Gallery
- Book now button

Instead, the experience should actively guide the visitor through the salon's services in a way that feels personal, visual, and easy to act on.

## Problem

Many hairstylists and salons rely on simple booking websites that do not properly sell the person, the skill, or the experience.

A visitor may land on the site and see a list of services, but they still have to figure out:

- Which service is right for them
- What the service means
- Why they should trust this stylist
- How quickly they can book

That creates friction.

## Solution

Create an interactive landing page where the stylist becomes the guide.

The main feature is a service semi-circle with the stylist positioned in the center or bottom of the visual system.

When a visitor selects a service, the website updates:

- The highlighted service
- The stylist pose or direction
- The service description
- The booking call-to-action

This creates a guided path:

**Choose a service → Understand it → Trust the stylist → Book**

## Primary Audience

The first version should focus on hairstylists and salons.

Secondary markets can include:

- Nail techs
- Brow artists
- Lash artists
- Barbers
- Massage therapists
- Estheticians
- Makeup artists
- Bridal beauty professionals

## Core Experience

The visitor should immediately understand:

1. Who the stylist/salon is
2. What they specialize in
3. What services are available
4. Which service is currently selected
5. How to book that service

## Design Personality

The design should feel:

- Clean
- Premium
- Spacious
- Warm
- Human
- Editorial
- Confident
- Easy to use

Avoid anything that feels cluttered, gimmicky, cheap, or overly animated.

## Business Value

This concept gives Mejor Tech a strong showcase project because it proves the company is not just creating websites, but designing experiences that make sense for the business.

For a salon, the site should not only look beautiful. It should help turn visitors into clients.

## Success Criteria

A successful version should:

- Make the salon owner/stylist feel central to the brand
- Show services in a creative but understandable way
- Work beautifully on desktop and mobile
- Make booking obvious and easy
- Feel different from a generic salon template
- Be reusable as a service-business website framework
